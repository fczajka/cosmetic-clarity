# Cosmetic Clarity - Mobile Cosmetic Ingredient Scanner

[![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=for-the-badge&logo=Flutter&logoColor=white)](https://flutter.dev) [![Dart](https://img.shields.io/badge/Dart-%230175C2.svg?style=for-the-badge&logo=Dart&logoColor=white)](https://dart.dev)

**Cosmetic Clarity** is a mobile application developed using Flutter that allows users to scan the ingredient lists of cosmetic products and view detailed information about each ingredient.

## Overview

This application aims to provide users with transparency regarding the ingredients in their cosmetic products. By simply scanning the ingredient list, users can quickly access information about each component, aiding them in making informed decisions about their purchases.

## Key Features

* **Ingredient Scanning:** Scan ingredient lists using the device's camera.
* **Detailed Ingredient Information:** Access comprehensive descriptions and details for each scanned ingredient.
* **History:** Review past scans with options to delete, edit, and view details.
* **Ingredient Database:** Search a complete database of cosmetic ingredients.
* **Personalization:** Mark ingredients as liked or disliked for future reference.
* **Settings:** Customize the app's theme and manage liked/disliked ingredients.
